# ngage-marketplace

ngage's personal Claude Code plugin marketplace.

## Installation

### Add the Marketplace

```bash
/plugin marketplace add nathan-gage/ngage-marketplace
```

Or if using a full URL:

```bash
/plugin marketplace add https://github.com/nathan-gage/ngage-marketplace.git
```

### Install a Plugin

Once the marketplace is added, install plugins with:

```bash
/plugin install react@ngage-marketplace
```
